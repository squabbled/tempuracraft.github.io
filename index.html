Welcome to TempuraCraft!

IP: mc.tempuracraft.com - Version: 1.20.2

TempuraCraft is a laid back Minecraft Java server running plugins meant to enhance the experience.

TempuraCraft was created to be what Minecraft servers were always supposed to be - communities formed around playing the game without pay-to-win, and being a space you can freely build and be creative.

The server has plugins such as mcMMO, SquareMap, and land-claim to allow you to build and explore as you feel fit.

What are you waiting for? Join the community today! Connect with the latest version of Minecraft Java to mc.tempuracraft.com
